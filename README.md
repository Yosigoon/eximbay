# eximbay
Eximbay API with python. Python 사용자를 위한 엑심베이 REST API 연동 모듈입니다.

- 이용 중 발생한 문제에 대해 책임지지 않습니다.
- 엑심베이에서 제공하는 문서와 예제를 보고 만들었습니다.


## 기능

1. 결제
2. 환불
3. 결제 내역 조회
